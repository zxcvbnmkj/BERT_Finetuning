# 微调 BERT 模型用于二分类任务
## 文件说明
- `main.py` : 训练代码
- `test_code.py` : 在有标签的测试集上测试模型性能
- `infer.py` : 在无标签数据上进行推理并保存正负样本的预测概率和最终预测结果
